# Laravel Movie Web App

 ### A Simple fullstack web app to get information about movies, tv-shows and actors from the TMDB API 
 
 ## To run copy the URL or download 
  - Run NPM Install 
  - Run "PHP Artisan serve" 
  - Go to 127.0.0.1/8000
